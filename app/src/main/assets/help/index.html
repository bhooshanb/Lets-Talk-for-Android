Let's Talk (Beta)

Why this app?

  • Are you tired of typing?
  • Are you fed up of auto-corrects & typos?
  • Want to send quick status update or instant message to friend?
  • If Yes,This is the right app for you!

How to use?

  • Start with Toggle Button. Blue Floating Circle (FC) will appear on Top-Left Corner.
  • On touching FC, Window will appear for the input. FC is available at any screen you are on.
  • Push the Green button to get your message.
  • A list of recognized text will appear as list.
  • Send appropriate text to send to current running app on Single Touch. Ex.: If Twitter is running in foreground, the selected text will directly go to Twitter's New-Tweet Window
  • If current foreground app doesn't accept text for sharing, the text won't be shared
  • For more sharing option, Long Press on message.

Things to remember!

  • The app is still in Beta, there are many more features are to be included in near future.
  • The App Permissions are required to run this perfectly. Please grant them with device-specific permission manager. Otherwise
  • We do not collect or save any user data to ourselves.
